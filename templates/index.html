About

In this project we chose a data set with macro and micro analysis related to geography. We then graph and present our data in a way that clearly presents the data in a way where it is easy to understand the story. I chose to do my project on crime rate data in the United States by state from 1960-2020.

One piece of feedback I recieved on my prototype was to make my macro page more organized and easier to follow. In my final version I will label my graphs with more descriptive headers and subheaders. Another piece of feedback I recieved is to organize my textboxes better. In my final version I will try and place my textboxes in ways that it is clear what they are reffering to.

    1. A data set with cime data on all 50 states was downloaded from The CORDIS project
    2. Google sheets was used to remove rows and columns that did not apply
    3. A useful dictionary-based data structure was designed in order to make all necessary data organized and accessible
    4. A Python program (formatData.py) was created to read the .csv line-by-line to create and save the dictionary as a .json file
All data from The CORGIS Dataset Project.
